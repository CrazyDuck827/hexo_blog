### Vue项目搭建打包流程

- 安装node.js

- npm淘宝镜像
```
    npm install -g cnpm --registry=https://registry.npm.taobao.org
```

- 安装全局 vue-cli
```
    cnpm install -g  vue-cli
```

- 安装依赖
```
    cnpm install
```

- 运行项目
```
    cnpm run dev
```

- 打包项目
```
    npm run build
    cnpm i && npm run build #解决打包慢 
```
